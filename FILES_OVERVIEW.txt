================================================================================
                            HOW TO USE EACH FILE
================================================================================

REACT APP DEVELOPMENT
  1. Edit src/App.tsx for UI changes
  2. Run: npm run build
  3. Preview in browser
  4. Check dist/ for built files

STREAMLIT APP
  1. Edit streamlit_app.py for features
  2. Run: streamlit run streamlit_app.py
  3. Refresh browser to see changes
  4. Access at http://localhost:8501

EDGE AI MODEL
  1. Prepare data in edge_ai_innovations/data/
  2. Run: python edge_ai_innovations/scripts/train.py
  3. Convert: python edge_ai_innovations/scripts/convert_tflite.py
  4. Inference: python edge_ai_innovations/scripts/infer_tflite.py

RASPBERRY PI DEPLOYMENT
  1. Run: bash edge_ai_innovations/scripts/raspberry_pi_setup.sh
  2. Copy models to Pi
  3. Run inference on device
